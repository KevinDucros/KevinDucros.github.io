FM Talk Foot est un générateur de podcast à partir de matchs du jeu Football Manager 2024. En plus de proposer une expérience inédite aux joueurs en ancrant leurs exploits dans le réel, le logiciel est totallement gratuit et sans pub. Sa particularité étant d'utiliser la puissance de calcul du pc de l'utilisateur pour générer le débat textuel et audio. Doté d'une interface accessible, il permet à tous les joueurs de garder un souvenir de leurs exploits !

Le défi technique

Innovation

Dataset créé en transcrivant de vraies émissions foot avec séparation automatique des locuteurs. Résultat : un modèle LLM personnalisé pour des débats cohérents qui sonnent authentique.

Architecture standalone

Application desktop autonome combinant PySide6 et FastAPI local.

Optimisation intelligente

Pipeline hybride : fine-tuning QLora avec Transformers, puis inference ultra-rapide via llama.cpp. Détection hardware automatique pour sélectionner le modèle GGUF optimal (Q4/Q8). Qwen-12B Q4 : seulement 4GB RAM, 3x plus rapide que Transformers.
